# CONTECNOW · Wacom Business Web

Web corporativa de CONTECNOW como partner oficial Wacom para empresas en España.

## Estructura

```
/
├── index.html        # Web principal (sectores, productos, testimoniales, contacto)
├── waas.html         # Página WaaS — Wacom as a Service (calculadora, comparativa, formulario)
├── assets/
│   ├── contecnow-logo.png
│   └── wacom-logo.gif
└── README.md
```

## Páginas

- **index.html** — Web corporativa completa: hero, estadísticas, 6 sectores, 3 productos, por qué CONTECNOW, testimoniales, formulario de contacto
- **waas.html** — Página dedicada al programa WaaS: calculadora interactiva de cuota mensual, tabla comparativa compra vs renting, formulario de solicitud de oferta

## Tecnología

HTML + CSS + JS vanilla. Sin dependencias externas excepto Google Fonts. Desplegable directamente en Netlify, GitHub Pages o cualquier hosting estático.
